# Electron Starter Template

An Electron starter template the following:
- Vite
- React
- Typescript
- Eslint
- Prettier
- Tailwindcss
- Shadcn configured

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Add shadcn components
```baseh
$ TS_NODE_PROJECT=tsconfig.web.json npx shadcn@latest add card
```

### Build

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
